Количество дней, за которое нужно уведомить администратора об окончании срока показа баннера — в этом поле можно указать за сколько дней до отключения баннера следует уведомить администратора сайта по почте. Актуально для тех баннеров, которым выставлен параметр "Дата окончания показа".
